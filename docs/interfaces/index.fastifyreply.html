Interface FastifyReply<RawServer, RawRequest, RawReply, RouteGeneric, ContextConfig>

FastifyReply is an instance of the standard http or http2 reply types. It defaults to http.ServerResponse, and it also extends the relative reply object.

Type parameters

  • RawReply: RawReplyDefaultExpression<RawServer> = RawReplyDefaultExpression<RawServer>
